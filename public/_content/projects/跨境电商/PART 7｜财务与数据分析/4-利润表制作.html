利润表制作

"Excel 是跨境电商最被低估的核心工具。一份能看清账的利润表，比任何选品工具都更影响你的生意。"

为什么要做利润表

没利润表的卖家

  • 不知道哪个 SKU 真赚钱
  • 不知道总成本到底多少
  • 凭"账户余额"判断业绩（错觉）
  • 关账时才发现"忙了一年没赚到"

有利润表的卖家

  • 每个 SKU 的盈利状况一目了然
  • 能找出"利润漏斗"在哪
  • 能精准决策（保哪个 SKU、砍哪个）
  • 能提前发现现金流危机

三种利润表

1. 单品利润表（最细）

每个 SKU 一行，计算单品净利

2. 月度 P&L（最实用）

店铺级月度损益表

3. 年度财报（最正式）

全年综合 + 同比环比

单品利润表（核心模板）

基础版（Excel 模板）

| 字段 | 公式 / 输入 |
|------|------------|
| **销售部分** | |
| 售价（含税） | 输入 |
| 销售佣金率 | 输入（如 15%） |
| 销售佣金 | = 售价 × 佣金率 |
| 净售价 | = 售价 - 佣金 |
| **平台费用** | |
| FBA 费 | 输入（如 $5.50） |
| 月仓储费均摊 | = 单件月仓储费 × 仓储月数 |
| 退货成本 | = 退货率 × 退货处理费 + 残值损失 |
| **物流成本** | |
| 头程运费均摊 | = (头程总费 / 总件数) |
| 关税 | = 申报价 × 关税率 |
| 汇损 | = 净售价 × 汇损率 |
| **营销成本** | |
| 广告费均摊 | = 售价 × ACoS × 广告订单占比 |
| **产品成本** | |
| 采购成本 | 输入（人民币） |
| 转外币 | = 采购成本 / 汇率 |
| 包装 / 配件 | 输入 |
| 其他摊销 | 输入（认证费、设计费等） |
| **计算结果** | |
| 总成本 | 各项相加 |
| 单品毛利 | = 净售价 - 直接成本 |
| 单品净利 | = 净售价 - 总成本 |
| 净利率 | = 净利 / 售价 |

示例填充

售价：$30.00
销售佣金率：15%
销售佣金：$4.50
净售价：$25.50

FBA 费：$5.50
仓储费均摊：$0.50
退货成本：$0.35
头程运费：$1.30
关税：$3.00
汇损：$0.50
广告费：$4.50
采购成本：$3.60（¥25 / 7.0）
包装：$0.30
其他摊销：$0.50

总成本：$20.05
单品净利：$25.50 - $20.05 + $4.50 = $9.95（注：广告/退货等已扣）
净利率：33% 

实际算法：
$30.00（售价）
- $4.50（佣金）
- $5.50（FBA）
- $0.50（仓储）
- $0.35（退货）
- $1.30（头程）
- $3.00（关税）
- $0.50（汇损）
- $4.50（广告）
- $3.60（采购）
- $0.30（包装）
- $0.50（其他）
= $9.45 净利
净利率：31.5%

月度 P&L（店铺级）

标准结构

收入项：
+ 销售收入（GMV）
- 平台佣金
- 退款损失
= 净销售收入

成本项：
- 产品成本（COGS）
- 物流成本（头程 + 派送 + 关税）
- 平台 FBA 费用
- 仓储费
- 退货处理
= 直接成本

毛利 = 净销售收入 - 直接成本

运营费用：
- 广告费
- 设计 / 内容费
- 软件订阅费
- 财税服务费
= 运营成本

EBITDA = 毛利 - 运营成本

固定费用：
- 工资 / 社保
- 办公租金
- 折旧摊销

净利润 = EBITDA - 固定费用

SKU 矩阵分析（实操工具）

维度

  • 销量
  • 毛利率
  • 净利率
  • 库存周转

四象限分类

            高毛利
              │
   ① 明星SKU  │  ② 现金牛
   （高利+高量）│ （高利+低量）
              │
低量 ─────────┼───────── 高量
              │
   ③ 长尾SKU  │  ④ 流量款
   （低利+低量）│ （低利+高量）
              │
            低毛利

行动建议

  • ① 明星 SKU：保护 + 扩展（出更多变体）
  • ② 现金牛：保护毛利，可适度涨价
  • ③ 长尾 SKU：砍掉 / 清仓
  • ④ 流量款：看是不是引流款，否则砍掉

现金流表（与利润表配套）

利润表只看"赚不赚"，现金流表看"钱在哪"。

简化版

期初现金
+ 销售回款（实际到账，不是 GMV）
- 采购付款
- 物流付款
- 广告付款
- 工资 / 房租 / 其他
+ 借款 / 融资入账
- 借款偿还
= 期末现金

详见 005-现金流管理。

Excel 模板的关键功能

1. 数据验证（避免乱输）

  • 净利率 < 0 标红
  • 库存 DOS < 30 标黄

2. 自动汇总

  • 数据透视表汇总 SKU 矩阵
  • VLOOKUP 关联多个表

3. 历史对比

  • 同比（去年同月）
  • 环比（上月）
  • 趋势图（最近 12 个月）

4. 预警机制

  • 净利率 < 15% 自动标红
  • 库存 < 30 天弹窗提示

利润表的"误差控制"

常见误差源

  1. 退货数据滞后：本月退货可能反映在下月数据
  2. 广告归因：广告订单 ≠ 广告带来的所有订单
  3. FBA 费用波动：实际 FBA 费 vs 预估有差
  4. 汇率波动：换算时点不同 → 不同利润

控制方法

  • 用"实际数据"而非"估算"
  • 每月对账（Amazon Statement vs 你的表）
  • 留 5% 误差缓冲

实战应用

  1. 新品上架前必填一次单品利润表——保底价 + 测试价
  2. 每月 1 号关上月账——做完整 P&L
  3. 建立"红黄绿"预警：核心指标超阈值自动标识
  4. 季度对账"实际 vs 测算"——找出预测偏差
  5. 利润表数据要"驱动决策"——看完要有 To-Do

进阶思考

利润表的本质是 "把你的生意翻译成数字"。

没有利润表的卖家：靠"账户余额"判断，滞后 + 失真 有利润表的卖家：实时知道"赚没赚、哪赚 / 哪亏"，主动 + 精准

真正的高手把利润表当"显微镜"+"放大镜"：

  • 显微镜：看每个 SKU、每个渠道、每个费用细节
  • 放大镜：从月数据看出年度趋势，提前 6 个月预判

一份优秀的利润表 ≈ 你的"经营操作系统"。 没有它，所有决策都是凭感觉。

思考题

  • 你目前有月度 P&L 吗？最近一次完整填写是什么时候？
  • 你能立刻说出你 Top 5 SKU 各自的净利率吗？
  • 你最赚钱的 SKU 和最不赚钱的 SKU，差距来自哪里？
