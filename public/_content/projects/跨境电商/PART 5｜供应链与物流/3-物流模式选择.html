物流模式选择

"物流模式不是'选哪个最便宜'——是'选哪个能让你赚到钱'。同一个产品用不同模式，利润率可能差 30%。"

三大模式核心对比

维度 FBA FBM（自发货） 海外仓
库存归属 Amazon 仓 卖家自有 第三方海外仓
发货执行 Amazon 卖家 第三方仓
流量加权 高（Prime + Buy Box） 低 中
客服承担 Amazon 卖家 部分第三方代
退货处理 Amazon（自动） 卖家（麻烦） 第三方代理
启动成本 低 极低 中
变动成本 高（FBA 费 + 仓储费） 中（物流费） 中（仓储 + 派送）
资金占用 高（库存在国外） 低 高
适合产品 标品、高周转 大件、定制、低周转 多平台 + 高退货

模式 1：FBA（Fulfillment by Amazon）

工作流程

1. 卖家发货到 Amazon FBA 仓
2. Amazon 入库、检验、上架
3. 用户下单 → Amazon 自动发货
4. 用户退货 → Amazon 处理 → 退到 FBA

FBA 的"真实成本"

项目 估算（小标准件 $30 售价产品）
FBA 履约费 $4-7/件
月度仓储费 $0.6-2/月/件
长期仓储费（>365 天） $6.9/立方英尺
退货处理费 $1-3/件
移除费 $0.5/件
弃置费 $0.15-0.30/件
总成本 约售价 20-30%

适合 FBA 的产品

  • 客单价 $15-100（FBA 费占比合理）
  • 体积适中（避开"大件附加费"）
  • 销量稳定（可备 60-90 天库存）
  • 退货率不高（< 8%）

不适合 FBA 的产品

  • 客单价 < $10（FBA 费占比太高）
  • 超大件（FBA 费可能 > $20/件）
  • 季节性极强（库存压死）
  • 高退货类（服装、定制类）

模式 2：FBM（Fulfillment by Merchant）

两种 FBM

  • FBM·中国直发：从国内直接发到美国买家
  • FBM·美国本土：从你的美国仓库发货

中国直发 FBM

  • 优势：库存在国内，资金占用少，可以备多 SKU
  • 劣势：时效 7-15 天（与 Prime 2 天差距大）→ CVR 低
  • 适合：客单价 > $50 + 用户能接受等待的品类

美国本土 FBM

  • 你在美国有自己的仓 / 朋友家里堆货 / 小型海外仓
  • 时效：3-5 天
  • 灵活性高，但管理麻烦

适合 FBM 的产品

  • 大件家具（FBA 费太贵）
  • 定制类产品（无法标准化入库）
  • 高利润 + 低周转（不必占 FBA 仓位）
  • 多 SKU 试销（备 FBA 风险高）

模式 3：海外仓（第三方仓）

工作流程

1. 大批量海运到第三方海外仓
2. 多平台对接（Amazon FBM / Shopify / eBay 都从此仓发）
3. 退货回到此仓 → 重新整理 → 二次销售

海外仓的核心价值

  • 多平台铺设：一仓服务 N 个销售渠道
  • 退货循环：高退货品类必备
  • 库存灵活：可以从海外仓再 "调拨" 到 FBA

海外仓的成本结构

项目 估算（小件）
入库费 $0.3-0.8/件
仓储费 $5-12/立方米/月
拣货 + 包装 $1-2/单
派送费（USPS / UPS） $4-8/包裹
退货处理 $1-3/件

选海外仓的"5 个 K"

  1. 位置：东西海岸都有覆盖的更好
  2. 派送时效：本土 3-5 天可达
  3. 系统对接：能 API 对接 Amazon / Shopify
  4. 退货能力：能否退货检测 + 二次销售
  5. 服务报价透明：避免"暗扣"

三模式组合策略

起步阶段

新品测款：FBM 中国直发 / 小批量 FBA
验证爆款：FBA 主仓
高退货款：海外仓

成熟阶段

爆款主流量：FBA（吃 Prime 流量）
长尾 SKU：海外仓（不占 FBA 仓位）
大件 / 定制：FBM

决策树

你的销量稳定吗？
├── 否（新品/低销）→ FBM 中国直发 + 少量 FBA 试水
└── 是（验证过的爆款）
    ├── 客单价 < $15 → 海外仓 + Amazon FBM
    ├── 客单价 $15-100 → FBA 主战
    └── 客单价 > $100
        ├── 大件 → FBM 美国本土
        └── 标品 → FBA + 海外仓备份

库存"三仓"分配（成熟卖家最佳实践）

70% 库存在 FBA：吃 Prime 流量
20% 库存在海外仓：FBA 断货时调拨 + 多平台分销
10% 库存在国内集货仓：随时补给海外

这种结构能在断货风险 + 资金效率 + 多平台之间最佳平衡。

常见误区

误区 1："新手必须先做 FBA"

  • 新品验证阶段，FBM 直发更经济
  • 跑通后再批量发 FBA

误区 2."FBA 太贵不划算"

  • FBA 贵，但流量加权 + Prime 加速带来的销售提升通常 > 成本
  • 同款产品 FBA vs FBM，FBA 销量经常是 FBM 的 3-5 倍

误区 3："海外仓只适合大卖家"

  • 中小卖家如果做多平台，海外仓是必备
  • 退货高的类目，海外仓性价比比 FBA 高

误区 4."一种模式打天下"

  • 单一模式 = 单一风险（FBA 关账户 / 海外仓倒闭 / FBM 物流时效）
  • 必须组合使用

实战应用

  1. 每个 SKU 单独做模式选择——不要"全部 FBA"
  2. 算清单 SKU 综合成本：包括仓储 / 退货 / 库龄
  3. 建立"模式切换"机制：销量变化、季节变化时可切换
  4. 海外仓选 2 家：一主一备，避免单点故障
  5. 季度复盘各模式 ROI：哪种模式利润最高？资源往哪倾斜？

进阶思考

物流模式的本质是 "用什么样的成本结构换什么样的销售机会"：

FBA = 高变动成本 + 高流量 FBM = 低成本 + 低流量 海外仓 = 中成本 + 中流量 + 灵活性

没有"最好的模式"——只有"和你的产品 / 阶段 / 战略最匹配的模式"。

做错模式选择的卖家，赚到的钱全交给了物流——表面 GMV 漂亮，实际利润为 0。

思考题

  • 你目前主要用哪种模式？这个模式对应的"隐形成本"你算清了吗？
  • 如果你的 FBA 突然爆仓限制入库，你的备份方案是什么？
  • 你的 SKU 矩阵里，有多少个用错了模式？切换后能省多少？
