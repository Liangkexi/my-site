# 跨境物流原理

> "一件商品从中国工厂到美国消费者手上，要经过 5 个交接点，每个交接点都要花钱、花时间、并且可能丢件。物流不是把货送到——是在时效、成本、风险三者之间做选择。"

## 物流链路全景

```
工厂 → 国内集货仓 → 头程运输（空/海/陆） → 目的国清关 → 海外仓/平台仓 → 尾程配送 → 消费者
  ①         ②              ③                  ④              ⑤              ⑥
```

每一段都对应不同的玩家、计费方式和风险点。

## 三大履约模式

| 模式 | 全称 | 谁存货 | 谁发货 | 适合场景 |
|------|------|--------|--------|---------|
| FBA | Fulfillment by Amazon | 亚马逊仓 | 亚马逊 | 标品、走量、爆款 |
| FBM | Fulfillment by Merchant | 卖家自有/海外仓 | 卖家 | 大件、定制、低周转 |
| 海外仓 | 第三方海外仓 | 第三方仓 | 第三方仓 | 多平台铺货、退换货高 |

**关键权衡**：FBA 流量加成 + 高仓储费；FBM 成本可控 + 时效劣势；海外仓兼顾，但管理复杂。

## 头程运输方式

| 方式 | 时效 | 成本 | 适用 |
|------|------|------|------|
| 空运（直飞） | 3-7 天 | 高（¥35-50/kg） | 高单价、补货救急 |
| 空派（空+派送） | 7-12 天 | 中（¥25-35/kg） | 中小件常规补货 |
| 海运（整柜/拼柜） | 30-45 天 | 极低（¥8-15/kg） | 备货、大宗 |
| 海派（海+派送） | 35-50 天 | 低 | 重货、铺货 |
| 中欧班列 | 18-25 天 | 中 | 欧洲市场 |

**经验法则**：销售预测稳定的 SKU 走海运备货，新品/旺季补货走空运。

## 清关：最容易踩坑的一环

### 三个核心要素
1. **HS Code 申报**：错误编码可能导致税率翻倍或扣关
2. **申报价值**：低报有罚款风险，高报税金虚高
3. **目的国合规**：FCC（美国电子）、CE（欧盟）、PSE（日本）等认证

### 常见清关模式
- **一般贸易**：合规、可抵扣增值税，但流程慢
- **快件清关**：DHL/UPS 自带清关，时效快但限品类
- **双清包税**：货代承担清关 + 关税，卖家拿"裸价"，最省心但灰色

## 运费构成（以海派为例）

```
头程费 = 重量计费（实重 vs 体积重，取大者）
      + 燃油附加费（10-25%）
      + 旺季附加费（Q4 普遍 +15%）
      + 偏远地区附加费（美国 ZIP 外）
      + 关税与清关费
```

**陷阱**：体积重 = 长×宽×高(cm) / 6000，泡货（轻而大）按体积重计费会贵到怀疑人生。

## 实战应用

1. **新品测款阶段**：FBM + 空运小批量，验证销量后再 FBA 备货
2. **爆款阶段**：海运备货 60-90 天用量到 FBA，断货 = 流量崩盘
3. **旺季前**：Q3 末提前发海运到位，Q4 临时缺货只能走空运（贵 3-5 倍）
4. **退换货**：高退货率类目（服装鞋包）必须配海外仓，否则退一件亏一件

## 常见问题

### 为什么 FBA 这么贵还有人用？
- 流量加权（Prime 标 + Buy Box 优先）
- 客服与退货由亚马逊处理
- 时效快带来的转化率提升通常 > 仓储费成本

### 海运为什么这么便宜？
- 集装箱单次运量大（一个 40HQ 柜≈ 28 立方米）
- 边际成本极低，按舱位拼柜可以摊薄到惊人
- 代价是 **资金被锁在路上 45 天**——这才是真正的成本

## 进阶思考

> 物流的本质不是"把货送到"，而是"把库存放在正确的位置"。
> 同一批货：放在工厂 = 死库存；放在海运船上 = 资金成本；放在 FBA = 销售机会。
> 每一次"发货决策"都是在替未来的销售下注。

## 思考题

- 你的产品如果毛利 30%，物流成本占售价的多少？砍掉 5% 物流费 = 多多少利润？
- 旺季前你应该用什么模型决定备货量？（提示：销售预测 × 历史季节系数 × 安全库存）
- 如果美国关税突然 +25%，你的供应链会怎么调整？海外仓 / 转口贸易 / 涨价转嫁？
