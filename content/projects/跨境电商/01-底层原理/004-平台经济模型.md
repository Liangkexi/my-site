# 平台经济模型

> "Amazon 不卖商品，它卖'流量'和'信任'。你交的不是佣金——是租金。理解平台是什么，才知道你在和谁博弈、为什么这个游戏的规则总在变。"

## 平台是什么

平台 = **撮合双边市场的基础设施**。

```
卖家 ←─── 平台（规则/流量/信任/履约） ───→ 买家
        抽佣 / 广告费 / 仓储费 / 数据
```

平台的商品不是货，而是 **匹配效率**：让一个想买无线耳机的人，遇见性价比最高的卖家——这件事本身值钱。

## 核心机制一：网络效应

### 双边正反馈
```
更多买家 → 卖家更愿意入驻 → SKU 更丰富 → 体验更好 → 吸引更多买家
   ↑                                                       ↓
   └───────────────────────────────────────────────────────┘
```

**结论**：平台天然倾向"赢家通吃"。一旦某品类领先，后来者很难追——这就是为什么 Amazon 在美国电商无可撼动。

### 对卖家的启示
- **大平台流量大但竞争惨烈**（Amazon、TikTok Shop）
- **小平台流量少但红利期长**（Temu 早期、TikTok Shop 早期）
- **入场时机**比"选哪个平台"更重要

## 核心机制二：流量分配

### 平台的"流量预算"是有限的
- 首页位、搜索前 3 位、推荐位 = 稀缺资源
- 平台用算法决定**谁能拿到流量**

### 三种分配逻辑
| 逻辑 | 典型平台 | 卖家应对 |
|------|---------|---------|
| 搜索匹配（关键词 + 转化率） | Amazon | Listing 优化 + 广告 |
| 推荐算法（兴趣 + 互动） | TikTok Shop | 内容创作 + 短视频 |
| 货架陈列（价格 + 销量） | Temu | 极致低价 + 快速迭代 |

### 算法的目标 ≠ 你的目标
- **平台目标**：GMV 最大化 + 用户留存
- **你的目标**：单量与利润
- 当两者冲突（比如算法奖励低价），平台永远赢

## 核心机制三：平台抽税（Platform Tax）

### 看得见的成本
| 项目 | Amazon | TikTok Shop | Shopify |
|------|--------|-------------|---------|
| 销售佣金 | 8-15% | 5-8% | 0%（自有店铺） |
| 履约费（FBA） | $3-7/件 | 由卖家自履约 | 自理 |
| 仓储费 | 月费 + 旺季溢价 | — | 自理 |
| 广告费 | ACoS 通常 15-30% | 内容流量为主 | Facebook/Google |
| 支付手续费 | 含在佣金中 | 1-3% | 2.9% + $0.30 |

### 看不见的成本
- **数据归平台**：销售数据是平台的，不是你的
- **客户归平台**：你不知道谁买了你的货
- **规则风险**：封号、降权、品类政策一夜变天
- **品牌稀释**：买家记得"我在 Amazon 买的"，不是"我在你这买的"

### 综合"平台税率"通常占 GMV 的 30-50%
> 这就是为什么独立站（Shopify）流量获取再贵，仍有人坚持做：**用流量成本换数据所有权和品牌资产**。

## 核心机制四：拍卖式广告

平台广告本质是 **关键词竞价拍卖**——你不是和平台谈价，是和其他卖家抢位。

```
你的广告排名 = 出价 × 质量分（CTR + 转化率 + 评分）
```

**推论**：
1. 早期高出价积累数据（哪怕亏钱）→ 提升质量分 → 后期低出价也能拿前位
2. **新品没历史数据 = 起跑就吃亏**，所以新品期烧广告很难躲
3. 同品类卷起来时，CPC 会无限上涨直到所有人都只剩微薄利润——这就是平台的"完美"商业模型

## 实战应用

1. **选平台 = 选生态**：你擅长投放就去 Amazon，擅长内容就去 TikTok，擅长低价供应链就去 Temu
2. **跨平台布局**：单一平台 = 单一风险源，至少 2 个渠道（一主一辅）
3. **数据沉淀双线走**：在平台跑量的同时，引流到独立站/邮件列表沉淀真实用户
4. **看透"红利期"**：新平台爆量阶段流量便宜——抓窗口期，等成熟了再进就只能交"平台税"

## 进阶思考

> 在平台上做生意有两个层次：
> **第一层**：把平台当渠道——卖货、收钱、走人。利润薄，永远在跟规则赛跑。
> **第二层**：把平台当用户获取入口——通过平台第一次接触客户，把他们沉淀到自己的池子（独立站/私域/品牌认知）。
>
> 第一层是"打工人"，第二层是"创业者"。区别在于：**离开这个平台你还剩下什么**。

## 思考题

- 如果 Amazon 明天突然把佣金提到 25%，你的业务能撑多久？有什么备选方案？
- TikTok Shop 现在像 2015 年的 Amazon——红利期能持续多久？怎么判断窗口关闭？
- 你今天卖的每一单，给你留下了什么"资产"？（库存？数据？复购客户？品牌？还是只剩一笔到账的钱？）
