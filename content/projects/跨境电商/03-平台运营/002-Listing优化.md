# Listing 优化

> "Listing 是你的销售员。它 7×24 小时工作，从不抱怨，但**它说什么、不说什么、用什么顺序说**，决定了访客是否变成订单。"

## Listing 的本质

一个 Listing 同时在做两件事：
1. **被搜到**（关键词覆盖）→ 决定流量
2. **被买单**（说服转化）→ 决定 CVR

这两件事经常冲突：堆关键词会让文案变蹩脚，纯文案派又会被埋没。**好的 Listing 是两者的平衡**。

## Listing 七大要素

```
1. 主图              ← 决定 CTR（最关键）
2. 标题              ← 决定搜索 + CTR
3. 副图（6-9 张）    ← 决定 CVR（场景/细节/对比）
4. 五点描述          ← 决定 CVR（卖点表达）
5. 价格              ← 决定 CVR 弹性
6. A+ 内容（图文）   ← 决定 CVR（信任 + 品牌）
7. Q&A + 评价        ← 决定 CVR（社会证明）
```

## 主图（CTR 决定者）

### Amazon 规则
- 白底（纯白 RGB 255,255,255）
- 产品占画面 85%+
- 不能有水印、文字、Logo（部分类目除外）
- 最少 1000×1000 像素（推荐 2000×2000）

### 突围方法
- **角度选择**：和 Top 10 不同的角度（多数白底正面 → 你做 45° 视角）
- **细节强化**：放大关键差异点（USB 接口、加大尺寸）
- **隐性场景**：人手拿着、对比尺寸（合规的方式）

### 主图 A/B 测试
- Amazon Manage Your Experiments 工具（需品牌备案）
- 至少跑 14 天
- CTR 提升 > 10% 才算赢

## 标题（搜索 + 转化）

### 黄金公式
```
品牌名 + 核心产品词 + 关键属性 1 + 关键属性 2 + 适用人群/场景 + 规格
```

### 示例（瑜伽垫）
```
YogaPro Premium Yoga Mat - 6mm Thick Non-Slip TPE Yoga Mat 
for Women Men, Eco-Friendly Exercise Mat with Carrying Strap 
(72"x24", Purple)
```

### 标题规则
- 长度：80-120 字符（最长 200，但 80-120 转化最好）
- 主关键词放前面（移动端只显示前 30 字符）
- 不要堆叠（不要"yoga mat exercise mat fitness mat"）
- 不要全大写、特殊符号

## 副图（CVR 杀手）

### 必备的 6 张
1. **场景图**：产品在使用中的样子
2. **尺寸图**：标注关键尺寸 + 对比物
3. **功能图**：每个卖点单独一张
4. **细节图**：材质、做工、关键部件特写
5. **对比图**：vs 普通版 / 旧款 / 同类
6. **包装图**：开箱时看到什么（送礼场景重要）

### 副图设计纪律
- **每张图回答一个购买疑虑**
- 加文字标注（Amazon 副图允许）
- 文字要大（移动端能看清）
- 一致的视觉风格

## 五点描述（说服的核心）

### 五点写作公式
**[利益点] + [证据] + [对比/数据]**

### 反面教材
```
❌ High Quality Material
   （没说什么材质、好在哪、跟谁比）
```

### 正面教材
```
✅ PREMIUM TPE MATERIAL - Our 6mm thick TPE yoga mat is 
   30% denser than standard PVC mats, providing superior 
   cushioning for your knees and joints during long sessions. 
   Free of harmful chemicals (SGS-certified, latex-free).
```

### 五点的优先级
- 第 1 点：最大卖点（差异化、消费者最在意的）
- 第 2 点：核心规格（尺寸 / 容量 / 材质）
- 第 3 点：使用场景或适用人群
- 第 4 点：风险消除（保修 / 退换 / 认证）
- 第 5 点：套装内容 / 礼盒信息

## 后台关键词（Search Terms）

### 规则
- 250 字节限制（约 250 个英文字符）
- 不要重复标题/五点已有词
- 不要写品牌名（自己的、别人的都不行）
- 用单数 / 复数变体覆盖

### 写作策略
- 长尾词：标题塞不下的关键词
- 错拼词：用户常打错的（如 "yogga mat"）
- 西语词：覆盖西语用户（约 12% 美国流量）
- 场景词：use case 相关词

## A+ 内容（品牌备案后）

### 模块组合
1. **品牌故事**：建立信任
2. **核心卖点轮播图**：把五点视觉化
3. **使用场景图**：让消费者代入
4. **对比表**：vs 同店其他款 / vs 竞品
5. **FAQ**：预先回答常见疑虑

### A+ 的隐性价值
- 提升 CVR 5-15%（Amazon 官方数据）
- 降低退货率（消费者预期更准）
- 替换"模板默认描述"，提升 SEO

## CVR 优化的优先级

```
影响 CVR 的因素，从大到小：
1. 价格 + 评价星级           ← 占 50%（短期内难改）
2. 主图                      ← 占 20%
3. 五点 + A+ 内容            ← 占 15%
4. 副图                      ← 占 10%
5. 标题（除了关键词覆盖）    ← 占 5%
```

**优化顺序**：先主图（快、见效快）→ 再 A+ 内容 → 再五点 → 最后标题微调。

## 实战应用

1. **上架前**：标题 / 五点 / 主图全部用 Top 3 对手做对标，别凭感觉
2. **上架后 30 天**：每周看 CTR + CVR，低于类目均值就改主图 / 价格
3. **每季度做一次"Listing 审计"**：检查所有要素是否仍然 best in class
4. **A+ 内容做一次值得**：好的 A+ 用 2-3 年不用大改
5. **评价 + Q&A 主动经营**：差评下方礼貌回复 + Q&A 主动答常见问题

## 进阶思考

> Listing 是一个**多变量优化问题**，但大多数人只优化"自己看得舒服"。
>
> 真正高 CVR 的 Listing 满足三个条件：
> 1. 在 3 秒内让访客知道 "这是什么"（主图 + 标题前 30 字）
> 2. 在 30 秒内让访客相信 "这适合我"（副图 + 五点）
> 3. 在 2 分钟内消除所有 "下单顾虑"（A+ + Q&A + 评价）
>
> 这三层任意一层断裂，CVR 就会塌掉。

## 思考题

- 你最近的 Listing CTR / CVR 各是多少？类目均值是多少？差距来自哪一层？
- 你的主图能不能让用户在缩略图（200×200）大小就看清产品？
- 五点里有多少句子是"自夸"，有多少是"给证据"？理想比例 = 1:3
