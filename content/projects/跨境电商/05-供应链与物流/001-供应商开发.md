# 供应商开发

> "你的供应商决定你的天花板。**找到一个一流供应商比找到一个爆款产品难 10 倍**——但前者能让你做 10 个爆款，后者只能做 1 个。"

## 供应商的"四个层级"

| 层级 | 特点 | 适合阶段 |
|------|------|---------|
| **贸易商** | 没有工厂，转手赚差价 | 起步测试（小批量） |
| **小型工厂** | 有产线，但规模小 | 小卖家主力（量 < 10K/月） |
| **中型工厂** | 标准化产能、合规体系 | 成熟卖家 |
| **大厂 / OEM** | 头部品牌代工厂 | 品牌阶段，做差异化 |

> **新手坑**：以为找到工厂直供就便宜，但小卖家在大厂连"客户"都不算，反而吃亏。**起步从贸易商或小工厂开始**。

## 找供应商的渠道

### 1. 1688（最主流）
- 阿里巴巴的国内批发平台
- 优势：选择多、议价空间大、起订量低
- 找法：
  - 搜产品关键词 → 按"实力商家"筛选
  - 看"30 天成交单数"和"回头率"
  - 警惕：图片质量极差、价格离谱低 → 多半骗子

### 2. 阿里巴巴国际站
- 主要面向海外买家，中文界面也能用
- 优势：合规体系强、有金牌供应商认证
- 劣势：报价高于 1688（因为有海外服务费）

### 3. 工厂展会
- 广交会（每年 4 月、10 月，广州）
- 义乌国际商贸城（小商品类目）
- 行业垂直展会（玩具展、电子展等）
- 优势：可以面对面验厂、看样品
- 适合：单次采购量 > ¥10 万

### 4. 工厂拜访
- 在 1688 找到候选 → 实地探访
- 看：厂区规模、设备、质检流程、工人状态
- 适合：要长期合作的核心 SKU

### 5. 同行推荐
- 跨境圈子的供应商口碑流传
- 优质供应商往往不主动推广，靠老客户介绍
- 加入卖家社群（公众号、微信群、知识星球）

## 供应商初筛"7 问"

第一次接触供应商，必问：

1. **MOQ（最小起订量）多少？**
   - 起步阶段 MOQ < 100 才友好
2. **打样费多少？打样周期多久？**
   - 标品打样应该免费 / 退还，定制收费 ¥200-2000
3. **生产周期多久？**
   - 标品 7-15 天、定制 20-45 天
4. **付款方式？**
   - 30% 定金 + 70% 见提单 = 行业标准
   - 要求 100% 预付 → 风险高
5. **质检流程？有出厂检吗？**
   - 没有出厂检的工厂直接 pass
6. **能否做 OEM / 改包装 / 印 Logo？**
   - 测他们的定制能力
7. **有合作过哪些跨境品牌？**
   - 看他们是否懂跨境电商需求（Listing 主图、英文标签等）

## 报价分析

### 价格构成（让供应商分项报价）
```
单价 = 材料成本 + 加工费 + 包装费 + 利润 + 验货费
```

### 议价的合理空间
- **报价砍 10-15%**：合理，工厂会接受
- **砍 20%+**：要么牺牲质量，要么对方撒谎报高
- **永远低于市场最低 30%**：99% 是骗局或质量陷阱

### 看穿"低价陷阱"
- 极低价 + "支持验厂" → 实际是贴牌商
- 极低价 + "送货上门" → 用次品凑数
- 极低价 + "即下即发" → 库存尾货 / 残次品

## 验厂（核心动作）

### 远程验厂（视频）
- 用微信视频让对方走一圈厂区
- 看：生产线在不在生产、工人多少、产品成品堆放
- 成本低，但能筛掉 80% 假工厂

### 实地验厂
- 提前不告知具体到访时间
- 看 6 个维度：
  1. 工厂规模（产能匹配）
  2. 设备先进度
  3. 仓库管理（5S 标准）
  4. 质检流程（QC 人员、检测工具）
  5. 工人专业度（熟练 vs 杂工）
  6. 老板气质（信不信得过）

### 第三方验厂
- 找 SGS / TUV / Bureau Veritas 等机构
- 成本：单次 ¥3000-10000
- 适合：年采购 > ¥100 万的核心供应商

## 供应商关系管理

### 建立"供应商档案"
每个供应商一个文件夹，记录：
- 基本信息（公司、联系人、地址）
- 产品线 + 报价历史
- 起订量、付款方式、生产周期
- 历次合作记录（数量、问题、评价）
- 替代供应商（备份名单）

### 长期合作的杠杆
- **定期返单**：每月固定下单 → 你成为他的稳定客户
- **预付优势**：肯付定金 / 肯先打款 → 议价权
- **量大优惠**：阶梯采购 → 单价下降
- **介绍生意**：帮供应商介绍其他卖家 → 换取优惠

### 供应商"备份"原则
- 核心 SKU 至少有 2 个供应商
- 主供 70%、备份 30%
- 备份不只是"出问题时切换"——是议价筹码

## 红色信号（立即 pass）

- ❌ 不愿意提供工厂照片 / 视频
- ❌ 不愿意签合同
- ❌ 要求 100% 预付
- ❌ 报价远低于市场
- ❌ 联系不稳定（经常换号 / 不回消息）
- ❌ 同一产品报价反复变化
- ❌ 拒绝出具营业执照

## 实战应用

1. **每个新 SKU 至少询价 5 家**——不要只问 1-2 家
2. **首次合作小批量试单**——MOQ × 1，不要大单
3. **建立"供应商记分卡"**：质量 / 交期 / 沟通 / 价格 / 配合度（每季度更新）
4. **重要 SKU 必须验厂**——视频验厂是底线，实地验厂是金标准
5. **付款一定走对公账户**——不要打个人微信 / 支付宝（出问题没法追责）

## 进阶思考

> 跨境电商的"产品差异化"很多卖家以为是 Listing 文案上的，
> 真正的差异化是 **供应链能力的差异化**：
>
> - 别人 30 天交货，你 15 天 → 你的库存周转快一倍
> - 别人 MOQ 1000，你能 200 → 你能跑更多新品
> - 别人质检合格率 90%，你能 99% → 你的差评少一半
> - 别人不能改包装，你能 → 你的品牌识别度高
>
> 这些都不在 Listing 上看得见，但**决定了你能不能走得远**。
> 找好供应商 = 找一辈子的合作伙伴，值得花 3 个月专门研究。

## 思考题

- 你目前的核心供应商是哪一类（贸易商 / 小厂 / 中厂 / 大厂）？合作多久了？
- 如果他们突然不能合作，你的备份供应商是谁？需要多久切换？
- 你最近一次实地验厂是什么时候？看到了哪些不能在视频里发现的问题？
