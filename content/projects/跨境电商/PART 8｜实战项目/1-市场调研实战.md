# 项目一：市场调研实战

> **目标**：选定 1 个目标品类，完成从宏观到微观的完整市场分析
> 
> **预计时间**：1-2 周
> 
> **预算**：$0（仅用免费工具和试用）

## 步骤

### 1. 宏观市场扫描
- [ ] 列出 3-5 个感兴趣的大品类
- [ ] 用 Google Trends 看趋势方向
- [ ] 用 Jungle Scout/卖家精灵看品类容量

### 2. 深度竞争分析
- [ ] 找出 Top 20 产品列表
- [ ] 分析价格带、评价数、评分分布
- [ ] 识别头部品牌的护城河

### 3. 需求验证
- [ ] 搜索量分析（关键词月搜索量）
- [ ] 季节性分析
- [ ] 是否有持续的需求信号

### 4. 利润测算
- [ ] 计算典型产品的完整利润
- [ ] 评估进入门槛

### 5. 输出最终报告
