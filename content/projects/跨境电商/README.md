# 跨境电商 — 从零到一

一个基于 **Karpathy 方法论**的跨境电商自学知识库。

## 方法论概览

| 阶段    | 核心      | 做的事情               |
| ----- | ------- | ------------------ |
| 第一性原理 | 理解底层机制  | 贸易、支付、物流、平台经济的物理定律 |
| 组件拆解  | 逐个攻破子系统 | 选品、运营、流量、供应链、合规、财务 |
| 系统集成  | 三步走实战   | 市场调研 → 上架产品 → 获取首单 |
| 逆向工程  | 向最好的人学习 | 分解 Top 卖家，提取可复用的策略 |

## 原则

- **动手 > 看教程**：每个知识点对应一个实操动作
- **输出 > 输入**：用自己的话写下来才算学会
- **闭环 > 完美**：先跑通最小模型，再优化
- **复利 > 速成**：每天积累一点点，比突击 10 小时有效
