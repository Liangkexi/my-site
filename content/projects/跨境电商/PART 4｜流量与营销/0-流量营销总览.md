# 流量与营销总览

> 在跨境电商里，流量不是"吸引"来的——是**购买**来的，区别只在于你是用钱买还是用内容买。

## 核心认知

平台上的流量分配是一个**拍卖市场**。理解这个市场如何运作，比学任何"技巧"都重要。

## 流量漏斗

```mermaid
flowchart LR
    A[曝光] --> B[点击]
    B --> C[加购]
    C --> D[购买]
    D --> E[复购]
    
    A -.->|影响因素| A1[关键词排名/广告出价]
    B -.->|影响因素| B1[主图/价格/评价]
    C -.->|影响因素| C1[Listing/视频]
    D -.->|影响因素| D1[购物车/配送速度]
    E -.->|影响因素| E1[产品质量/客服]
```

## 关键指标

- **ACOS** = 广告花费 ÷ 广告收入
- **CTR** = 点击量 ÷ 曝光量（衡量主图+价格吸引力）
- **CVR** = 订单数 ÷ 点击量（衡量 Listing 转化能力）
