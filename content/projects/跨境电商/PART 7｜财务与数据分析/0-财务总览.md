# 财务与数据分析总览

> 跨境电商的本质是一门**数学题**。如果算不清账，赚的钱可能只是数字游戏。

## 核心认知

大部分新手卖家的死亡原因不是没订单，而是**算不清账**——表面"赚"了，实际上库存积压、广告费侵蚀、退货吞噬了所有利润。

## 利润公式

```
单品净利润 = 售价 
           - 平台佣金 (15% ± 5%)
           - 物流费 (FBA/FBM)
           - 产品成本 
           - 广告费 (ACOS × 广告订单占比)
           - 退货损耗 (退货率 × 退款率)
           - 汇损 (1% - 3%)
```

## 定价倍率参考

| 售价区间 | 理想倍率 | 说明 |
|----------|---------|------|
| $10-20 | 3-5x | 低价品靠量，对成本极敏感 |
| $20-50 | 3-4x | 黄金价格带，竞争最激烈 |
| $50-100 | 2.5-3.5x | 利润空间大，但广告成本高 |
| $100+ | 2-3x | 高客单价，转化率低 |
