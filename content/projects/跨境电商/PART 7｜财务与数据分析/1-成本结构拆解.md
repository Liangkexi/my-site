# 成本结构拆解

> "卖家的真相不是'我的产品赚了多少'——是'我**实际**赚了多少'。两者差距可能 50% 以上。**算不清账的卖家，赚的都是数字幻觉**。"

## 完整成本结构（5 大模块）

```
跨境电商总成本 = 
    产品成本（COGS）
  + 物流成本（Shipping）
  + 平台费用（Platform Fees）
  + 营销费用（Marketing）
  + 固定费用（Overhead）
```

每一块都有"显性 + 隐性"两部分。

## 模块 1：产品成本（COGS）

### 显性成本
- 工厂采购价（EXW / FOB）
- 国内集货 / 仓储费

### 隐性成本
- **样品成本**：选品时买的样品（¥3000-30000/品类）
- **打样费**：定制 / 改款的开模费
- **包装设计费**：彩盒 / 说明书设计
- **拍照建模费**：主图 / A+ / 视频
- **认证费**：CE / FCC / FDA 等
- **首件检验费**：第三方 QC

### 隐性成本的"摊销"
- 不要一次性吃在第一批货上
- 按"前 1000-3000 单"摊销
- 例：首批认证 ¥10K → 预计卖 2000 单 → 每单分摊 ¥5

## 模块 2：物流成本

### 显性成本
- 头程运输（空 / 海 / 铁）
- FBA 派送费（亚马逊扣）
- 海外仓派送费

### 隐性成本
- 关税 + 进口增值税
- 燃油 + 旺季 + 偏远附加费
- 保险费
- 报关 / 清关代理费
- 退货回程物流

### 计算口径
**单件物流总成本 = (头程费 + 关税 + 派送费 + 退货摊销) / 总件数**

## 模块 3：平台费用

### Amazon 费用结构

| 费用 | 金额 / 比例 |
|------|-----------|
| **销售佣金（Referral Fee）** | 8-15%（按类目） |
| **FBA 履约费** | $3-7/件（按尺寸重量）|
| **月度仓储费** | $0.6-2/月/件 |
| **长期仓储费**（>365 天）| $6.9/立方英尺/月 |
| **移除费 / 弃置费** | $0.15-0.50/件 |
| **退货处理费** | $1-3/件 |
| **专业卖家月费** | $39.99/月 |

### 其他平台抽佣对比
- TikTok Shop：5-8% 销售佣金
- Shopify：2.9% + $0.30 支付费 + $29-299/月套餐
- eBay：10-15% 销售佣金

## 模块 4：营销费用

### 站内广告（最大块）
- SP / SB / SD 广告花费
- 衡量：ACoS（建议 < 25%）+ TACoS（建议 5-15%）

### 站外推广
- KOL 合作费
- Deal 站折扣损失
- Google / Facebook 广告
- 社媒账号运营成本

### 隐藏的"营销成本"
- 主图 / Listing 反复优化的设计费
- A+ 内容制作
- 视频制作

## 模块 5：固定费用（Overhead）

### 经营成本
- 公司注册 + 维护
- 财税服务费
- 软件订阅（Helium 10、Keepa、ERP）
- 工具订阅（设计、AI、办公）

### 团队成本
- 工资 / 社保
- 培训
- 团建

### 资金成本
- 银行手续费
- 信用卡分期利息
- 商业贷款利息
- **机会成本**：占用资金的年化收益

## 单品利润计算（完整版）

### 输入数据（例）
```
售价：$30.00
平台佣金率：15%
FBA 费：$5.50
单件月仓储费：$0.50
广告 ACoS：25%
广告订单占比：60%
退货率：5%
退货处理费：$2/件
单件采购价：¥25
头程运费：¥8
关税：10%
汇率：7.0
其他摊销：¥3/件
汇损：2%
```

### 计算
```
1. 销售收入：$30.00
2. 平台佣金：$30 × 15% = -$4.50
3. FBA 费：-$5.50
4. 仓储费均摊：-$0.50
5. 广告费：$30 × 25% × 60% = -$4.50
6. 退货成本：5% × $2 + 5% × $5 残值损失 = -$0.35
7. 汇损（2%）：-$0.60
8. 总收入扣完平台 / 营销 / 退货：$14.05

转人民币：$14.05 × 7.0 = ¥98.35

成本：
9. 采购：¥25
10. 头程：¥8
11. 关税（10% × $30 × 7.0）：¥21（注：实际按申报价）
12. 其他摊销：¥3
13. 总成本：¥57

单品净利润：¥98.35 - ¥57 = ¥41.35
净利率：¥41.35 / ¥210 (即 $30×7) = 19.7%
```

> **健康基准**：净利率 ≥ 20%。 < 15% 危险。

## 容易"被遗忘"的成本

### 1. 长期仓储费陷阱
- 库龄 > 365 天 → $6.9/立方英尺/月
- 一个普通箱子（1 立方英尺）放一年 = 浪费 $80+
- 比销毁还贵

### 2. 测款失败成本
- 测 10 个 SKU，成功 1-2 个
- 失败的 SKU 也要"算入"总成本

### 3. 起始投资摊销
- 公司注册、商标、网站建设
- 按 3-5 年摊销

### 4. 时间成本
- 自己作为创始人的"工资"
- 每月给自己算 ¥10K-30K（机会成本）

### 5. 旺季"扛单"成本
- 旺季前 60 天的备货资金占用
- 临时空运补货的高成本

## 成本视角的"两个层面"

### 单品视角（micro）
- 每个 SKU 单独算成本 + 利润
- 用于：选品、定价、SKU 取舍

### 店铺视角（macro）
- 所有 SKU + 平台费用 + 团队 + 营销
- 用于：整体利润、估值、退出

> 单品赚钱 ≠ 店铺赚钱
> 多个 SKU 都微赚，但加上 Overhead 可能整体亏

## 实战应用

1. **建立"单品成本模板"**——每个 SKU 都跑一遍完整计算
2. **每月做"店铺 P&L"**——按"销售 - 直接成本 - 间接成本 - 固定费用"分层
3. **追踪"隐性成本占比"**——目标 < 20%
4. **每季度复盘"长尾 SKU"**——亏钱的果断砍掉
5. **建立"成本预警线"**：净利率 < 15% 立即分析原因

## 进阶思考

> 财务的本质不是"记账"——是 **"用数字看清自己"**。
>
> 看似简单的"卖一单赚多少钱"，背后涉及 30+ 个变量。
> 每个变量小波动 → 利润大波动。
>
> **3 类卖家的区别**：
> 1. 不算账的：感觉赚钱，实际可能亏（最危险）
> 2. 算大账的：知道大概赚多少，但不知道哪里漏
> 3. 算细账的：每分钱都清楚，能精准优化
>
> 顶级卖家不是"卖得最多的"，是 **"账算得最清的"**。
> 因为只有清楚成本结构，才能找到"放大利润的杠杆点"。
