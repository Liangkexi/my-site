# 定价策略

> "定价是跨境电商最被低估的杠杆。**调价 1% 通常影响利润 5-10%**——这是任何'优化主图''调广告'都难以达到的效率。"

## 定价的"四种思路"

### 1. 成本加成定价（错误的起点）
```
售价 = 成本 × N 倍
```
- 优点：简单
- 缺点：忽视市场承受力、忽视竞争对手
- **新手坑**：算出来的价 vs 市场价差很远 → 卖不动 / 不赚钱

### 2. 竞争对标定价（最常用）
```
售价 = Top 5 竞品中位数 ± 10%
```
- 优点：保证你在"消费者心理价格带"内
- 缺点：忽视自身成本结构 / 差异化价值
- 适合：标品、红海类目

### 3. 价值定价（高阶）
```
售价 = 消费者愿付的最高价 - 留余地
```
- 关注：消费者感知到的价值（不是成本）
- 适合：差异化 / 创新产品
- 难点：需要测试 + 数据支撑

### 4. 阶梯定价（动态）
- 新品期：偏低（冲销量 / 评价）
- 成长期：标准价
- 成熟期：可微涨（享受品牌溢价）
- 衰退期：折扣清仓

## 价格带的"心理学"

### 锚定效应
- Listing 上显示"原价 $40 → 现价 $25" → 消费者感觉"赚了 $15"
- 即使原价是虚构的，效果仍然有

### 99 心理
- $29.99 vs $30 → 第一个看起来"更便宜"
- 跨境定价多用 $X.99、$X.97

### 价格档位
- $X9 / $X4 是常见档位
- 跨过整十数（$30 → $31）感觉跳价大

### 价格定位的"信号"
- 高于均价：品质 / 品牌信号
- 接近均价：性价比 / 主流定位
- 低于均价：低价 / 清仓信号

> 你想给消费者"什么样的第一印象"？价格是最直接的信号。

## 定价的"具体流程"

### Step 1：摸底市场
- 看 BSR Top 10 价格分布
- 找出"价格带集中区"（如 $20-30 集中了 70% 卖家）
- 注意：避开"价格血战区"，找"相对空旷区"

### Step 2：核算保底价
```
保底价 = 单件总成本 / (1 - 最低可接受净利率)

例：
单件总成本 ¥70，最低净利 15%
保底价 = ¥70 / 0.85 = ¥82 = $11.7（汇率 7）
```
**这是你的"绝对底线"**——低于此价别卖。

### Step 3：测试价格弹性
- A/B 测试：±10%、±20% 不同价位
- 看 CVR + 销量 + 总利润
- 找到"利润最大化"价格点

### Step 4：动态调价
- 旺季前：可适度涨价
- 滞销期：折扣促销
- 竞品大降价：跟还是不跟（看你的成本结构）

## 价格 vs CVR vs 利润的"三角"

### 错误思路：追求"最低价"
- 价格越低 → CVR 越高（短期看）
- 但毛利空间越小
- 总利润可能反而下降

### 正确思路：找"最大利润价"
- 不是 CVR 最高的价（可能太便宜）
- 不是单品利润最高的价（可能销量太少）
- 是 **"销量 × 单品利润"最大值**

### 示例
```
价格 $25：CVR 12%，月销 300 单，单品利润 $5 → 月利润 $1500
价格 $30：CVR 9%，月销 200 单，单品利润 $8 → 月利润 $1600 ✅
价格 $35：CVR 6%，月销 130 单，单品利润 $12 → 月利润 $1560
```

**关键**：测试不同价格点 → 找拐点。

## 涨价的"艺术"

### 何时可以涨价
- BSR 排名稳定 + 评价数 500+
- 自然流量占比 > 50%
- 差评率 < 5%
- 库存充足

### 涨价幅度
- 一次 5-10%（小步）
- 不要一次涨 20%+（CVR 会断崖）

### 涨价的"伪装"
- 主图换"升级版"标识
- A+ 内容更新（强化新价值）
- 配套增加（套装 / 配件）

### 涨价后的指标观察
- 14 天观察期
- CVR 下降 > 30% → 回滚
- CVR 下降 10-20% → 接受（利润仍上升）

## 降价的"陷阱"

### 危险信号
- "对手降价我也降"陷阱：导致价格战
- "促销引爆"陷阱：习惯了低价，正常价 CVR 暴跌

### 健康的降价
- **清滞销**：库龄 > 180 天的果断折扣
- **季节性**：旺季后清库存
- **新品冲量**：上架 30 天内打折拿评价

## Coupon 和折扣的"细节"

### Amazon 几种促销
- **Coupon**：消费者可领，扣款式折扣
- **Lightning Deal**：限时秒杀（需达资格）
- **Best Deal**：日常促销
- **Subscribe & Save**：订阅省钱（消耗品）

### 选哪种？
- Coupon：日常引流 + 提 CVR（推荐）
- Lightning Deal：冲量 / 清滞销（旺季前）
- S&S：高复购品类必选

### Coupon 折扣率
- 5-10% 最常用
- 15-20% 强力但稀释品牌
- > 25% 看起来"出问题"（消费者怀疑）

## 套装定价（拉客单价）

### 套装设计逻辑
- 主品 + 配件：客单价高 + 减少配件复购成本
- 多件装：批量买更便宜（如 2-pack / 4-pack）
- 礼盒装：节日场景，溢价 30-50%

### 套装定价公式
```
套装价 < 单买各件总价（让消费者感觉"省了"）
但
套装毛利率 ≥ 单买毛利率 1.1 倍（你也要赚）
```

## 实战应用

1. **每个新品上架前算 3 个价**：保底价 / 市场价 / 高端价
2. **每月做一次"价格弹性测试"**——别永远固定一个价
3. **建立"调价日历"**：旺季前 / 黑五前 / 季节交替必查
4. **跟踪 Top 10 竞品价格**——他们调价你要知道
5. **不要为"赢得一个差评"而降价**——单点退让会被算法捕捉

## 常见误区

### 误区 1."低价就能卖得多"
- 低价 = 利润薄 + 平台不爱（低 Buy Box 加权）
- 低价竞争是死循环

### 误区 2."我成本就是这么多，不能降"
- 真正的定价基于市场，不基于你的成本
- 如果市场价低于你的成本 → 优化供应链，不是硬卖

### 误区 3."调一次价后不动了"
- 价格是动态变量
- 至少每季度复审

### 误区 4."Coupon 越大越好"
- 大 Coupon 拉来"价格敏感"用户，无品牌忠诚
- 适度（5-10%）就好

## 进阶思考

> 定价的本质不是 **"卖多少钱"**——是 **"给消费者什么样的信号"**。
>
> 同样的产品：
> - 卖 $15 → 信号："便宜货 / 凑合用"
> - 卖 $30 → 信号："标准品 / 主流选择"
> - 卖 $80 → 信号："高级品 / 品质保证"
>
> 消费者不只是买"产品本身"——买的是"产品 + 自我认同"。
> 一个买 $80 瑜伽垫的人，是在买"我重视健康 / 我有品味"的感觉。
>
> **定价不是数学题，是心理学题**。
> 顶级品牌的定价权来自"他们让消费者愿意付溢价"——而不是"成本加成"。

## 思考题

- 你目前的产品定价是怎么定的？依据什么？
- 你最近一次价格 A/B 测试是什么时候？结果如何？
- 如果让你今天把所有产品价格 +10%，你的总利润会怎么变？
