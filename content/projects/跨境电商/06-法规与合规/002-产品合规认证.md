# 产品合规认证

> "认证不是'加分项'——是'入场票'。**没认证就上架 = 赌博**：今天没事，明天可能下架 + 罚款 + 库存销毁。"

## 为什么必须做认证

### 法律层面
- 美国：CPSC、FDA、FCC 等强制要求
- 欧盟：CE 标志强制（无 CE 不可入境）
- 日本：PSE 标志（电子类）

### 平台层面
- Amazon 各品类有"合规审核"
- 不达标 → 强制下架
- 严重者 → 暂停销售权限

### 责任层面
- 用户因产品受伤 → 可能被起诉
- 个人责任无限（小卖家通常没有公司隔离）

## 主要市场认证速查

### 美国

| 品类 | 认证要求 | 颁发机构 |
|------|---------|---------|
| 电子电气 | FCC | FCC |
| 食品接触 | FDA | FDA |
| 药品 / 化妆品 | FDA 注册 | FDA |
| 儿童产品 | CPSC + CPSIA | CPSC |

| 个人防护用品（PPE）| OSHA | OSHA |
| 节能电器 | Energy Star | EPA |

| 品类 | 认证 | 备注 |

## CE 标志（欧盟最常见）

### 误区
- ❌ "CE 是欧盟的认证机构发的"——其实没有 CE 机构，是**自我声明**
- ❌ "贴个 CE 标就行"——必须做对应测试

### 正确流程
1. 确定适用指令（EMC、LVD、RED 等）
2. 找第三方实验室做测试
3. 出具 DoC（Declaration of Conformity）
4. 留存技术文件
5. 产品上贴 CE 标志

### 成本
- 简单产品：¥3000-10000
- 复杂电子：¥20000-100000

### 防伪 CE
- 真 CE：字母间距标准
- 假 CE（"中国出口" CE）：间距不同——欧盟不认

## FCC（美国电子）

### 适用范围
- 含无线发射的设备（蓝牙、WiFi、遥控）
- 电子设备（即使无无线，可能也要 SDoC 自我声明）

### 三种 FCC
- **Verification（自我验证）**：低风险设备
- **DoC / SDoC（自我声明）**：通用电子
- **Certification（认证）**：含无线发射的设备（最严）

### 成本
- SDoC：¥3000-8000
- Certification：¥15000-50000

## CPSC + CPSIA（美国儿童产品）

### 强制要求
- 12 岁以下儿童产品都要做
- 第三方实验室测试
- 出具 CPC 证书（Children's Product Certificate）
- 永久标识（可追溯）

### 重点检测项
- 铅含量（< 100ppm）
- 邻苯二甲酸盐
- 小零件（窒息风险）
- 锋利边缘

### 成本
- 一个 SKU 测试：¥3000-15000
- 不同款式 / 颜色 / 材质都要分别测

## RoHS（限制有害物质）

### 适用
- 欧盟所有电子电气产品
- 限制 10 种有害物质（铅、汞、镉等）

### 成本
- ¥1500-5000/批次

## 食品接触认证（FDA / EU 1935）

### 适用
- 餐具、杯子、保鲜盒、宠物碗等
- 任何"会接触食物"的产品

### 测试项目
- 重金属溶出
- 总迁移量
- 特定物质迁移
- 异味 / 异色

### 成本
- 单个材质：¥2000-8000

## 认证的"操作流程"

### Step 1：确定认证清单
- 根据产品 + 目标市场 → 列出所有需要的认证
- 找咨询公司 / 第三方实验室帮你梳理

### Step 2：找第三方实验室
- 知名机构：SGS、TUV、Intertek、Bureau Veritas
- 国内便宜的：华测、广电计量
- 对比报价 + 周期

### Step 3：送样测试
- 工厂提供样品
- 实验室测试 1-4 周

### Step 4：拿报告 + 证书
- 通过 → 拿到证书 + 测试报告
- 不通过 → 改产品 → 复测

### Step 5：上架准备
- Listing 上传证书
- 包装贴标签（CE、FCC 等）
- 留存所有文件备查

## 认证成本核算

### 单 SKU 起步认证清单（电子产品 - 美国市场）
| 项目 | 成本 |
|------|------|
| FCC SDoC | ¥5000 |
| RoHS（如出欧盟） | ¥3000 |
| 加州 65 警示 | ¥2000 |
| **小计** | **¥10000** |

> 这只是"基础合规"——一些复杂产品可能 ¥50K+。
> **没预算就别选合规复杂的品类**。

## 平台合规审核（Amazon）

### 哪些品类会被审？
- 儿童产品（必审 CPC）
- 化妆品（必审 FDA 注册）
- 食品接触（必审）
- 电子（FCC ID 备查）
- 玩具（CPSIA + ASTM）
- 体重秤、灯具等（特定标准）

### 审核流程
1. 上架时上传相关认证文件
2. Amazon 审核 1-7 天
3. 通过 → 正常销售
4. 不通过 → 需要补充 / 修改

### 没通过怎么办
- 看清楚要补什么文件
- 找实验室补测试
- 重新提交（不要硬扛）

## 常见误区

### 误区 1："小卖家没事，没人查"
- 平台抽检越来越严
- 用户投诉触发审查
- 一旦被查，连带历史订单都查

### 误区 2."认证一次永久有效"
- 标准更新会失效（2-5 年要更新）
- 产品改版要重新测
- 不同批次可能要复测

### 误区 3."认证报告随便弄个就行"
- 现在 Amazon 验证报告真实性
- 假报告 = 永久封号
- 必须真实可查的实验室出具

### 误区 4."别的卖家没做我也不做"
- 抽检是"概率事件"
- 别人没出事不代表你也不会
- 出事 = 全军覆没

## 实战应用

1. **选品阶段就考虑合规成本**——别等做出来才发现要 ¥50K 认证
2. **建立"产品认证档案"**——所有证书 + 报告云存档
3. **找 1 家长期合作的检测机构**——熟悉你的产品类目
4. **关注法规更新**——欧盟标准每年变
5. **优先选低合规门槛的品类起步**——服装、家居小物 vs 电子、化妆品

## 进阶思考

> 合规是 **"小卖家的劣势，大卖家的护城河"**。
>
> 小卖家觉得合规贵 → 不做 → 上架被拒 / 被罚
> 大卖家把合规当门槛 → 投入 → 形成壁垒（小卖家进不来）
>
> 高合规门槛的品类 = **天然的低竞争市场**。
> 选这种品类，前期辛苦投入，后期享受"无人能进"的红利。
>
> 合规不是成本——是 **"用钱筛掉竞争对手"** 的工具。

## 思考题

- 你目标市场 + 品类，需要哪些认证？总成本多少？
- 你的产品认证齐全吗？过期了吗？
- 你愿意进入"高合规门槛"的品类吗？这是机会还是负担？
