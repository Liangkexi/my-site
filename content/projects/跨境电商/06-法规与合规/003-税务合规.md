# 税务合规

> "税务问题不会立刻杀死你的店铺——但它会在你**最赚钱的时候**找上门，连本带罚地拿走利润。**逃税不是节省成本，是延后付费 + 利息**。"

## 跨境电商面对的"四种税"

| 税种 | 谁征 | 谁缴 | 何时缴 |
|------|------|------|--------|
| **进口关税** | 目的国海关 | 进口方 / 卖家 | 货物入境时 |
| **VAT / GST**（增值税） | 目的国税务局 | 卖家 | 销售后定期申报 |
| **销售税**（美国）| 各州税务局 | 卖家 / 平台 | 销售后定期申报 |
| **企业所得税** | 注册地税务局 | 卖家公司 | 年度申报 |

## 美国销售税

### 历史背景
- 2018 年前：仅在"有实体存在"的州才需缴
- 2018 年后（Wayfair Case）：销售达到一定门槛就要缴
- 各州门槛不同（一般 $100K 销售或 200 笔订单）

### 平台代缴（Marketplace Facilitator Tax）
- Amazon / eBay / Walmart 在大多数州**已经代缴**
- 卖家无需直接申报销售税
- 但需要在某些州注册"销售税编号"

### 独立站（Shopify）
- 不享受平台代缴
- 必须自己申请 + 申报
- 用 TaxJar、Avalara 等工具自动化

### 实战建议
- Amazon 卖家：基础阶段不用太焦虑销售税（平台代缴）
- Shopify 卖家：必须了解 + 自动化工具

## 欧盟 VAT（增值税）

### 适用范围
- 在欧盟销售商品给消费者（B2C）
- 标准税率：英国 20%、德国 19%、法国 20%、西班牙 21%

### 注册门槛（旧规则）
- 销售达到一定金额需要注册
- 跨境销售门槛不同

### IOSS（One-Stop Shop）2021 年新规
- < €150 货值：可走 IOSS（一次申报覆盖所有欧盟国）
- > €150 货值：在每个销售国分别注册 VAT

### VAT 注册流程
1. 找税务代理机构（每个国家）
2. 提交公司资料 + 销售数据
3. 拿到 VAT 号（1-3 个月）
4. 月度 / 季度申报

### VAT 注册成本
- 单国 VAT 注册：€500-2000/年（含申报代理）
- IOSS 注册：€500-1500/年

### 不注册 VAT 的代价
- Amazon EU 强制要求（不注册不能卖）
- 罚款 + 补缴 + 利息
- 海关扣货

## 中国税务（出口端）

### 出口退税
- 国内供应商出口可退增值税
- 你买货不能直接退（除非自有出口资质）
- **影响**：选有出口资质的供应商，价格更优

### 跨境电商综合税
- 个人邮包：行邮税（部分免税额）
- 9810 / 9710 海关编码：跨境电商出口政策
- 自营出口需要外贸公司资质 / 跨境电商企业资质

## 收款的税务处理

### 收款方式
- **第三方收款**（Payoneer / 万里汇 / PingPong）
  - 卖家从平台收外币
  - 然后通过收款机构换成人民币入账
  - **关键**：换汇过程必须合规（不要"地下钱庄"）

### 个人收款 vs 公司收款
- **个人收款**：早期卖家常用，但额度有限
- **公司收款**：必须有外贸 / 跨境电商资质
- **建议**：起步阶段月销 < ¥10 万可用个人，超出后必须公司

### 收款的税务申报
- **个人**：累计达到一定金额要申报个税
- **公司**：销售额计入营收，缴企业所得税
- **不申报**：收款机构有义务上报税务局，迟早被查

## 公司架构（跨境卖家常见）

### 模式 1：纯个人户
- 优点：简单
- 缺点：税务风险大、无法做大、无法品牌备案
- 适合：起步测试阶段

### 模式 2：国内有限公司
- 注册一家国内公司
- 通过公司收款 + 申报
- 适合：年销 ¥100 万以下

### 模式 3：香港公司
- 离岸收款方便
- 利得税相对低
- 适合：有海外多平台 / 多国销售

### 模式 4：美国公司（LLC / Inc）
- 注册美国公司收款
- 适合美国主战场卖家
- 复杂：需了解美国税法

### 模式 5：BVI / 开曼
- 离岸架构，避税效率最高
- 复杂 + 成本高
- 适合：年销 $1M+ 大卖家

## 跨境电商的"灰色地带"

### 现状
- 早期卖家普遍税务不合规（平台未严格审核）
- 2020 年后逐步收紧

### 风险演化
- **平台层面**：要求税务编号、自动汇报销售数据
- **国内层面**：金税四期，对个人外汇收入加强监控
- **国际层面**：CRS 全球税务信息交换

### 合规化的"早做晚做"
- **早做**：起步就规范，长期省心
- **晚做**：以后被查 + 补税 + 罚款 + 利息（成本翻倍）

## 税务合规的"三步走"

### 阶段 1：起步期（< $10K/月）
- 个人收款（控制额度）
- 留存收支凭证
- 心里清楚：这是临时阶段，要尽快升级

### 阶段 2：成长期（$10K-100K/月）
- 注册国内公司 / 香港公司
- 公司收款
- 建立基础财务记账
- 申报销售税（如适用）

### 阶段 3：成熟期（$100K+/月）
- 完整公司架构
- 专业税务顾问
- VAT / 销售税完全合规
- 年度审计

## 实战应用

1. **从 Day 1 留存所有收支记录**——日后有用
2. **VAT 注册要提前**——欧盟卖家先做 VAT 再做 EU 站
3. **找懂跨境的税务代理**——不要找传统会计师
4. **公司架构早规划**——做大后再改成本极高
5. **关注政策变化**——税务规则每年都在收紧

## 常见误区

### 误区 1."Amazon 帮我代缴了，我不用管"
- 平台代缴只是"销售税"——还有 VAT、所得税
- 而且代缴 ≠ 你不需要注册税号

### 误区 2."用个人账户收款没人查"
- 个人账户外汇收入累计 5 万美元 / 年要申报
- 银行有义务上报到税务局

### 误区 3."公司注册了就高枕无忧"
- 注册 ≠ 合规
- 必须按时申报 + 缴税 + 审计

### 误区 4."等查到我再处理"
- 一旦被查 = 至少 3 年回溯
- 罚款 + 滞纳金 + 补税，金额惊人

## 进阶思考

> 跨境电商的税务问题，本质是 **"全球化生意 + 各国分割税收"** 的天然矛盾。
>
> 你在中国生产、用 Amazon 卖到美国、收款到香港、最终汇到国内——
> 这条链上每一环都有税收主体在主张管辖权。
>
> 试图"全部躲过"几乎不可能（CRS、金税四期都在收紧）
> 试图"完全合规"成本极高（多个国家专业代理）
>
> 现实做法是 **"在合规和成本之间动态平衡"**：
> - 小规模时：相对粗糙，但留好记录
> - 中规模时：核心节点合规（VAT、公司架构）
> - 大规模时：全合规 + 专业团队
>
> 关键是 **不要在某一规模上"卡死"**——主动升级合规能力，跟上业务节奏。

## 思考题

- 你目前的收款方式是什么？合规吗？月度规模到了升级临界点了吗？
- 如果你做欧盟市场，VAT 注册了吗？哪个国家？
- 如果税务局明天来查你过去 3 年的销售记录，你能拿出完整凭证吗？
