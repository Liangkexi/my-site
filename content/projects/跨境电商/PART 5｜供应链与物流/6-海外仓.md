# 海外仓

> "海外仓不是 FBA 的'平替'——它是 FBA 解决不了的问题的解。**没有海外仓的卖家，做不了多平台、做不了大件、做不了高退货品类**。"

## 海外仓的"独特价值"

### FBA 解决不了的问题
1. **多平台库存共享**：FBA 的库存只能给 Amazon，不能发 Shopify / eBay
2. **退货后处理**：FBA 退货只能销毁 / 移除，海外仓可以二次销售
3. **大件成本**：FBA 大件费爆炸，海外仓更友好
4. **库龄成本**：FBA 长期仓储费极高，海外仓便宜
5. **灵活操作**：贴标、组合、改包装等 FBA 不允许

### 海外仓的核心场景
- 多渠道分销（Amazon FBM + Shopify + eBay 同库存）
- 退货中转（FBA 退货 → 海外仓 → 二次销售）
- 大件 / 重货存储
- 长期备货（不怕长期仓储费）
- FBA 中转（大批量先到海外仓 → 分批发 FBA）

## 海外仓的"四种类型"

### 1. 自建仓
- 你自己租仓库 + 雇人
- 优势：完全控制
- 劣势：成本高、管理复杂
- 适合：年 GMV > $5M 的大卖家

### 2. 第三方专业海外仓
- 如：易境通、谷仓、4PX 海外仓
- 优势：专业、系统对接好
- 劣势：成本中等、灵活性受限
- 适合：90% 中小卖家的最佳选择

### 3. 半托管海外仓
- 第三方提供仓储 + 你来管运营
- 优势：可控 + 不重资产
- 适合：成长期卖家

### 4. 公共云仓（共享）
- 多个卖家共用一个仓
- 优势：成本最低
- 劣势：响应慢、灵活性差
- 适合：起步测试

## 选海外仓的"7 大维度"

### 1. 地理位置
- 美东（新泽西、纽约）：覆盖东海岸快
- 美西（洛杉矶、加州）：覆盖西海岸 + 太平洋来货
- **理想**：东西海岸都有覆盖

### 2. 仓库规模 + 稳定性
- 看公司成立时间（< 3 年风险高）
- 看现有客户数量
- 看实地仓库照片 / 视频

### 3. 系统能力
- 能否 API 对接 Amazon / Shopify / eBay
- 能否实时同步库存
- 能否自助下单 / 查报告

### 4. 派送能力
- 自有派送 vs 对接 USPS / UPS / FedEx
- 时效（本土 3-5 天可达）
- 偏远地区覆盖率

### 5. 退货处理能力
- 能否签收退货
- 能否拍照 / 检测 / 二次入库
- 能否换标 / 翻新

### 6. 服务报价
- 入库费、仓储费、操作费、派送费
- 看有没有"暗扣"（清单要详细）

### 7. 应急能力
- 旺季爆仓时能不能保你的货
- 系统故障时的应对

## 海外仓的成本结构

### 入库环节
| 项目 | 估算 |
|------|------|
| 卸柜费 | $200-400/柜 |
| 入库操作（按 SKU） | $0.3-0.8/件 |
| 标签贴付 | $0.2-0.5/件 |
| 上架费 | $0.1-0.3/件 |

### 仓储环节
| 项目 | 估算 |
|------|------|
| 仓储费 | $5-15/立方米/月 |
| 长期仓储费（>180 天） | +30-50% |
| 温控仓 | +50-100% |

### 出库环节
| 项目 | 估算 |
|------|------|
| 拣货费 | $0.5-1.5/单 |
| 包装费 | $0.5-1.0/单 |
| 派送费（USPS） | $4-8/包裹 |
| 派送费（UPS） | $6-12/包裹 |

### 其他
| 项目 | 估算 |
|------|------|
| 退货签收 | $1-2/件 |
| 退货检测 | $1-3/件 |
| 退货二次入库 | $0.5-1/件 |
| 弃置费 | $0.2-0.5/件 |

## FBA + 海外仓的"组合拳"

### 组合 1：FBA 主战 + 海外仓后备
```
70% 库存在 FBA（吃 Prime 流量）
30% 库存在海外仓（FBA 断货时调拨）
```
**适合**：大部分中小卖家

### 组合 2：海外仓主战 + FBA 借势
```
80% 库存在海外仓（多平台共享）
20% 库存在 FBA（用 Amazon FBM 卖，借 Prime 标提高 CVR）
```
**适合**：多平台分销 / 高退货品类

### 组合 3：海外仓中转 → FBA
```
大批量海运到海外仓 → 按需分批转入 FBA
```
**优势**：避免一次性入太多到 FBA（IPI 风险）

## 海外仓的"管理税"

### 隐形成本
- 多套库存系统对接
- 报表合并复杂
- 财务对账麻烦
- 出错时责任界定

### 应对工具
- ERP 系统：店小秘、马帮、易仓
- 库存预警：自动监控 FBA + 海外仓 + 在途

## 退货处理（海外仓的杀手锏）

### FBA 退货的痛
- 不可二次销售（退到 FBA 不重新上架）
- 移除费 + 销毁费高
- 信息不透明

### 海外仓退货的优势
1. 退到海外仓 → 拍照检测
2. 完好品 → 重新换标 → 二次上架
3. 轻微瑕疵 → 翻新 / 折价销售
4. 严重损坏 → 销毁

### 退货回收率
- 完好品比例：30-50%
- 翻新可售比例：20-30%
- 销毁比例：20-30%
- **综合回收率**：40-70%（vs FBA 退货 0% 回收）

> **示例**：年退货 1000 件，单价 $20，海外仓比 FBA 多回收 50% = $10,000 净增。

## 自建海外仓 vs 第三方仓

| 维度 | 自建 | 第三方 |
|------|------|--------|
| **启动成本** | 高（$50K+） | 低（按使用） |
| **固定成本** | 高（房租 + 人员） | 无（按量收费） |
| **灵活性** | 完全控制 | 受限 |
| **专业度** | 自己摸索 | 现成专业 |
| **规模成本** | 量大后省钱 | 量大后贵 |
| **适合阶段** | 年 GMV > $5M | < $5M 全部用第三方 |

## 实战应用

1. **起步用第三方海外仓**——不要轻易自建
2. **选海外仓签 1 年合同 + 试用 3 个月**——表现好再续
3. **永远有 2 家海外仓**：主一备一，分散风险
4. **建立"FBA - 海外仓 - 国内仓"统一库存视图**
5. **退货高的 SKU 全部走海外仓**——不要让 FBA 处理

## 常见误区

### 误区 1."只有大卖家才用海外仓"
- 错。多平台 / 高退货 / 大件，从小卖家就该用

### 误区 2."海外仓比 FBA 便宜"
- 综合算下来不一定（取决于流量加权差距）
- 但**特定场景下**性价比远超 FBA

### 误区 3."海外仓不够稳定"
- 选靠谱的就行（看历史 + 规模 + 客户口碑）
- 比 FBA 更不会"突然封号"

### 误区 4."海外仓自动管好库存"
- 你必须自己监控库存、补货、调拨
- 海外仓只是"执行方"，不是"运营方"

## 进阶思考

> 海外仓的真正价值不是"另一个仓库"——是 **"你在海外有了根据地"**。
>
> 没有海外仓的卖家是"游击战"：货到 FBA → 卖完 → 重新发货
> 有海外仓的卖家是"阵地战"：海外有库存 → 任何渠道随时调拨 → 退货可二次销售
>
> 这是从"短期生意"进化到"持续经营"的关键基础设施。
> Amazon 大卖家几乎 100% 都有海外仓——不是因为"省钱"，是因为 **"灵活性 = 竞争力"**。

## 思考题

- 你目前用海外仓吗？如果用，是哪家？满意度如何？
- 你的退货品有多少能"二次销售"？海外仓能帮你回收多少？
- 多平台分销是你 1-2 年内的目标吗？海外仓是这条路的起点
